# MRO Lite

MRO Lite is a simplified Maintenance, Repair and Overhaul (MRO) system designed to manage maintenance jobs for aircraft and equipment.

The goal of this project is to simulate a real-world MRO system using enterprise-level technologies, following clean architecture and best practices.

## Tech Stack

### Backend
- C#
- ASP.NET Core (.NET 8)
- Entity Framework Core (Code First)
- SQL Server (Docker)

### Frontend
- Angular
- TypeScript
- Angular Material
